## 📂 Featured Projects

### 🔹 [USD/IDR Exchange Rate Volatility Forecasting](https://github.com/lulukainullatifah/usd-idr-exchange-rate-forecasting)
**Tools:** R (rugarch, tseries, ggplot2, FinTS, xts)
**Methods:** ARMA, TARCH, EGARCH, ADF Test, Sign Bias Test, MLE (BFGS)

Forecasted Rupiah exchange rate volatility using 979 daily observations
(Jan 2021 – Dec 2024) from Bank Indonesia. EGARCH(2,1) was selected as
the most robust model based on parameter stability (Nyblom test) and
confirmed asymmetric leverage effect.

**Key Finding:** Negative shocks (Rupiah depreciation) produce significantly
stronger volatility responses than positive shocks of equal magnitude.
